## Usage

1. Install dependency
`ansible-galaxy install -r requirements.yml`

2. Make sure user can run sudo without password (e.g Edit `/etc/sudoers.d/`) or use `sudo whoami` to cache the token
  
3. `ansible-playbook main.yml`
